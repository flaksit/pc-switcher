<objective>
Classify apt collateral before refusing it (D-30). Today `_converge_install`/`_converge_remove` refuse any install whose simulation would remove or downgrade anything not itself reviewed, and `_collect_plan_time_collateral` surfaces collateral as `REPORT_ONLY` facts the user cannot act on. That blanket refusal blocks a legitimate install whose only collateral is a dependency nobody chose.

Purpose: the corrected D-30 splits collateral by provenance. Auto-installed collateral (a dependency apt pulled in, absent from `apt-mark showmanual`) is apt doing its job and proceeds silently. Manually-installed collateral is something the user chose to have, so it becomes its own reviewable item offering install-anyway, skip, or abort. The question belongs in the batched review, decided at plan time, never as a prompt during apply.

Output: plan-time collateral classification that emits an actionable review item for manual collateral only; a review-group shape that carries the three-way install-anyway / skip / abort choice; and an apply-time guard that honours the recorded decision instead of refusing unconditionally.
</objective>
